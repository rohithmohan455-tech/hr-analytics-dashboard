# HR Analytics Dashboard

## Project Overview
The HR Analytics Dashboard provides insights into employee data to help organizations understand workforce trends, employee distribution, and attrition patterns. The dashboard enables HR teams to make data-driven decisions regarding employee management and organizational planning.

## Objective
The goal of this project is to analyze HR data and visualize key workforce metrics using Power BI. The dashboard helps monitor employee statistics such as department distribution, employee count, and attrition rates.

## Tools & Technologies
- SQL
- Excel (Data Source)
- Tableau
- Data Modeling
- Data Visualization

## Dataset
The dataset contains employee information including:
- Employee ID
- Department
- Job Role
- Age
- Gender
- Salary
- Attrition Status
- Years at Company

The data was imported from Excel and transformed for analysis.

## Key Features
- Total Employee Count
- Attrition Rate Analysis
- Department-wise Employee Distribution
- Gender Distribution
- Average Salary Insights
- Employee Age Distribution

## Data Processing Steps
1. Imported HR dataset from Excel.
2. Cleaned and structured the data for analysis.
3. Created data relationships and calculated metrics.
4. Built interactive dashboards and visualizations in Power BI.

## Dashboard Insights
The dashboard provides HR teams with the ability to:
- Identify departments with higher attrition
- Analyze workforce demographics
- Monitor employee distribution across roles
- Evaluate salary trends and employee tenure
